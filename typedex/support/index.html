Contact

All feedback and requests for support can be directed to us via email.



Minimum requirements

TypeDex requires Chrome 83, Edge 83, Opera 68, Safari 13, and Firefox 77. It may work on older browsers but performance is not guaranteed.



Data Inaccuracies

Pokémon data is entered by hand to ensure it is quickly up to date. Unfortunately from time to time this can result in small inaccuracies. If you find any, please report them. It's much appreciated.
