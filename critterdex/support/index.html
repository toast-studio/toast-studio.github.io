Contact

All feedback and requests for support can be directed to us via email.



Requirements

CritterDex requires Chrome 83, Edge 83, Opera 68, Safari 13, and Firefox 77. It may work on older browsers but performance is not guaranteed.



FAQ

Why was Catch Boost removed in CritterDex 1.2?

The people who datamine the game have removed that from their data set, so it was also removed from CritterDex. They believe it was used in previous games but not New Horizons.


Data Inaccuracies

Data is sourced from a third party. If you find any inaccuracies, please report them. It's much appreciated.



Translations

If you find any inaccuracies with translations please submit corrections via this Github project. If you don't have a Github account, please submit them via email. All corrections are welcome and appreciated. Once updated your name will be added to the Acknowledgements section of the app.
